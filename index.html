JSON教程

json是Javascript对象的一种，
也是制作Addons必不可少的一部分，本教程主要针对Addons的制作

json的语法格式简单，要求比较不严格，本教程将会采用图文类比的方法，使得内容易于理解 json文本一般有五种格式:
字符串 "a":"b"
数值 "a": 1
布尔值 "a": true
数组 "a": [1, 2]

先来看一段例子
{"教程": {"内容": "json","作者": "炖菜"}}
这就是一段json文本。其中"教程"是父标签，"内容"和"作者"都是子标签，用图片表示的形式如下

所以从某种层面上来说，json可以理解为有一个最大的文件夹，包着里面的一个个小文件夹，小文件夹同样也可以放更小的文件夹在里面
而对于"a": 1或者"a": true这种规定了具体的值的标签，可以类比为文件，存放在文件夹中。 但是对于一段比较长的json文本，我们通常需要换行和进退位来使其更美观，例如我们可以把以上例子换成
{
"教程": {
"内容": "json",
"作者": "炖菜"
}
}
值得注意的是，同一级标签之前要用英文逗号隔开，否则会使语法出错

穷举器

用法：列举出"NXY有1个肝"到"NXY有9个肝"中的整数
"NXY有"(该文本放入第一框)
"1"(该文本放入变量框)
"个肝"(该文本放入第二框)
并设置变量最大值为9，穷举间隔为1

第一框(选填)
变量框(必填，只支持数值)
第二框(选填)

变量最大值
穷举间隔(填0网页崩溃)
